Example

Using the following table as input:

ItemCost
Apple0.25
Banana0.25
Orange0.40
Fruit hip holster10.00

...the following table results if 'Item' and 'Cost' are specified as column names:

CostItem
Apple0.25
Banana0.25
Orange0.40
Fruit hip holster10.00

Please note that only the header line is modified.
